This document

This document is the specification for University Radio York's BAPS3 playout suite. It is written with the intention of being compilable using rustbook.

It is, currently, a living specification. It will change dramatically and without warning, as BAPS3 is designed and implemented. Hopefully, when the reference implementations are finished, the specification will be frozen and usable as a reference for future BAPS3 work.
