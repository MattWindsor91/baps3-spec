Bifrost

Bifrost Protocol

This chapter of the specification defines the Bifrost protocol. This is the set of conventions Bifrost-compatible clients and servers use to communicate.

Structure

The Bifrost protocol is split into several stages:

  • the Message Protocol, which defines the basic protocol on top of TCP used by Bifrost;

  • the Core Protocol, which defines the structure and content of messages (requests and responses) used in Bifrost clients and servers;

  • the Roles, which define packages of requests and responses that define part of a Bifrost system.
